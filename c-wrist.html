C-Wrist

About

The C-Wrist is a wearable vital monitoring system designed to alleviate hospital overcrowding by enabling at-home care for mild COVID-19 outpatients. This project engineered an efficient device to assess critical parameters—body temperature, heart rate, and oxygen saturation—leveraging electrical device theory, sensor integration, and signal conditioning. By providing real-time data, it empowers physicians to prioritize care without compromising quality, addressing a pressing need in respiratory illness management.

Task/Action

  • Developed 3D CAD models in SOLIDWORKS, designing electrical enclosures with cable routing, button interfaces, wiring linkages, hinges, and wristwatch components, optimized for injection molding, sheet metal fabrication, and precision machining.
  • Managed project budgets, delivered weekly technical reports, and applied engineering principles to ensure cost-effective development.
  • Performed failure mode analysis, root-cause investigations, and corrective design iterations to enhance component reliability.
  • Designed and executed Design of Experiments (DOEs) to refine and down-select configurations, achieving optimal sensor performance.
  • Integrated Arduino and Python for real-time data analysis, improving measurement accuracy by 15% through advanced signal processing.

Result

The C-Wrist emerged as a robust wearable system for monitoring mild COVID-19 outpatients, tracking body temperature, heart rate, and oxygen saturation—key indicators in respiratory health. Utilizing a SparkFun Pulse Oximeter and Heart Rate Sensor, an M5STACK NCIR Temperature Sensor, and an accelerometer for motion correction, it delivers four metrics via an LCD display in real time. Patients receive alerts for critical vitals, enabling safe daily activity with a 15% improvement in data reliability over baseline designs.

Tech Stack: SOLIDWORKS, Arduino, Python, Sensor Integration
