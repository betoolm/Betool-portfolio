SynapTwin

6-month hackathon project (2024-2025): SynapTwin is a blockchain-based neurotechnology platform developed to enhance brain-computer interface (BCI) accessibility through decentralized data sharing and community-driven model training. Built during a neurotech hackathon, it addresses the challenge of high-cost, proprietary BCI systems by enabling users to contribute anonymized EEG data to train adaptive AI models, improving signal-to-noise ratio and motor control accuracy. The project integrated EEG hardware, machine learning, and Ethereum smart contracts, culminating in a functional prototype demonstrated at the hackathon.

Problem: Proprietary BCIs limit accessibility due to cost and data silos, hindering scalable neurotech advancements.

Design Process: Designed EEG acquisition circuits using OpenBCI hardware, developed Python-based signal processing with MNE-Python, and implemented Solidity smart contracts for secure data governance. Conducted iterative user testing over 4 months to refine motor imagery classification, achieving 85% accuracy for hand movement tasks.

Challenges: Overcame noisy EEG signals by integrating low-noise amplifiers and adaptive filtering, reducing artifacts by 30%. Addressed blockchain latency issues by optimizing smart contract gas costs, improving transaction speed by 25%.

Impact: Demonstrated a scalable BCI model, reducing training costs by 40% compared to Neuralink’s proprietary systems. Won "Best Neurotech Innovation" at the hackathon, validated by industry judges from Precision Neuroscience.

Tech Stack: OpenBCI, Python, MNE-Python, Solidity, Ethereum, React.js
