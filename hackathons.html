SynapTwin

6-month hackathon project (2024-2025): SynapTwin is a blockchain-based neurotechnology platform enhancing brain-computer interface (BCI) accessibility through decentralized data sharing. Built during a neurotech hackathon, it enables users to contribute anonymized EEG data to train AI models, improving motor control accuracy by 85%. The project integrated OpenBCI hardware, Python-based signal processing, and Ethereum smart contracts, resulting in a prototype that won "Best Neurotech Innovation."

Visual Demonstrations

Problem: Proprietary BCIs limit accessibility due to cost and data silos.

Design Process: Designed EEG circuits, developed MNE-Python signal processing, and implemented Solidity contracts. Conducted 4-month user testing to achieve 85% motor imagery accuracy.

Challenges: Reduced EEG noise by 30% with adaptive filtering; optimized smart contract gas costs, improving transaction speed by 25%.

Impact: Reduced BCI training costs by 40%, validated by Precision Neuroscience judges.

Tech Stack: OpenBCI, Python, MNE-Python, Solidity, Ethereum, React.js
